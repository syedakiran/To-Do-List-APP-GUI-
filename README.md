# To-Do List App (GUI)

##  Project Overview

The **To-Do List App (GUI)** is a C++ based graphical application designed to help users manage their daily tasks efficiently. Unlike console-based programs, this application provides an interactive and user-friendly interface where users can easily add, view, update, and delete tasks.

This project is suitable for **students, beginners, and academic assignments** who want to understand GUI-based application development in C++.

---

##  Technologies Used

* **Programming Language:** C++
* **GUI Framework:** wxWidgets / SFML (depending on implementation)
* **Compiler:** MinGW (GCC)
* **IDE:** Dev-C++ / Code::Blocks / Visual Studio (optional)

---

##  Features

* Add new tasks
* View all tasks in a list
* Edit existing tasks
* Delete completed or unwanted tasks
* Simple and clean graphical user interface
* Beginner-friendly code structure

##  Installation & Setup

### Prerequisites

* C++ Compiler (MinGW recommended)
* GUI library installed (wxWidgets or SFML)

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/To-Do-List-APP-GUI.git
   ```
2. Open the project in your preferred IDE
3. Configure the GUI library paths (include & lib)
4. Compile and run the project
